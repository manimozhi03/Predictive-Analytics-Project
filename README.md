## Predictive Analytics Using Historical Data

## Project Overview
This project predicts future sales trends using historical sales data with linear Regression.

## Features
-Data preprocessing using Pandas
-Date conversion and month extraction
-Linear Regression model
-Sales prediction
-Model accuracy evaluation
-Sales visualization using Matplotlib

## Technologies Used
-Python
-Pandas
-Matplotlib
-Scikit-learn
-Excel

## Files

## Output

-Sales predictive using Linear Regression
